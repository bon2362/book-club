Колонка «Приоритет» показывает № X из N — место этой книги среди всех записей пользователя.
Не расставившие — в отдельной группе снизу, заметно выделены серым.

Итоговые решения

  • Без приоритетов — отдельная группа снизу таблицы с пометкой «Не расставили приоритеты»
  • Подсказка в личном кабинете — оранжевый баннер, исчезает после первого перетаскивания
  • Приоритеты — в существующей вкладке «Участники», новая колонка + фильтр по книге
  • Формат приоритета: «№ X из N» — сразу видно контекст (1 из 3 важнее, чем 1 из 12)
