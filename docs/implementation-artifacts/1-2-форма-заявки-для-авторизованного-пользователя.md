# Story 1.2: Форма заявки для авторизованного пользователя

Status: done

## Story

As an authenticated user,
I want to fill and submit a book proposal form with all required fields,
So that I can propose a book for the club catalog and receive confirmation.

## Acceptance Criteria

1. **[AC1 — Поля формы]** Когда форма открыта → она содержит все 9 полей: Название (required), Тема (dropdown), Писатель (required), Число страниц, Дата издания, Ссылка на текст, Описание, Ссылка на обложку, Почему стоит прочитать (required)

2. **[AC2 — Темы из кэша]** Дропдаун тем содержит варианты из каталога Google Sheets (получены через props из уже загруженного кэша, без дополнительных запросов к Sheets API)

3. **[AC3 — Success state]** После успешной отправки отображается: "Заявка принята! Мы рассмотрим её в ближайшее время." — внутри того же модального окна (форма заменяется сообщением, модал не закрывается)

4. **[AC4 — Валидация]** При отправке с незаполненными обязательными полями → форма показывает ошибки валидации рядом с каждым пустым обязательным полем

5. **[AC5 — Mobile UX]** Форма не требует горизонтальной прокрутки; поля вызывают подходящий тип мобильной клавиатуры (text, number, url); доступна через временную кнопку "Предложить книгу" в BooksPage для залогиненных пользователей

### Валидация onBlur

```typescript
const [errors, setErrors] = useState<Record<string, string>>({})

function validateField(name: string, value: string) {
  if (['title', 'author', 'whyRead'].includes(name) && !value.trim()) {
    setErrors(e => ({ ...e, [name]: 'Обязательное поле' }))
  } else {
    setErrors(e => { const next = { ...e }; delete next[name]; return next })
  }
}
```

### Submit handler

```typescript
async function handleSubmit(e: React.FormEvent) {
  e.preventDefault()
  // Validate all required fields first
  const newErrors: Record<string, string> = {}
  if (!title.trim()) newErrors.title = 'Обязательное поле'
  if (!author.trim()) newErrors.author = 'Обязательное поле'
  if (!whyRead.trim()) newErrors.whyRead = 'Обязательное поле'
  if (Object.keys(newErrors).length > 0) { setErrors(newErrors); return }

  setStatus('submitting')
  try {
    const res = await fetch('/api/submissions', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ title, topic: topic || undefined, author,
        pages: pages ? Number(pages) : undefined, publishedDate: publishedDate || undefined,
        textUrl: textUrl || undefined, description: description || undefined,
        coverUrl: coverUrl || undefined, whyRead }),
    })
    if (!res.ok) throw new Error('Submit failed')
    setStatus('success')
  } catch {
    setStatus('error')
  }
}
```
