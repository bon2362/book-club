# Глоссарий

Короткие определения терминов, которые часто встречаются в проекте.

| Термин | Значение |
| --- | --- |
| Runtime | То, что реально используется работающим сайтом прямо сейчас. |
| Next.js | Фреймворк, на котором построены страницы и API. |
| NextAuth | Библиотека авторизации. |
| Drizzle | TypeScript-инструмент для работы с Postgres и схемой БД. |
| Neon | Облачный Postgres. |
| Vercel | Хостинг Next.js-приложения. |
| GitHub Actions | Автоматические проверки и jobs после push. |
| Allure | Красивый отчет по e2e-тестам. |
| Codecov | Сервис контроля покрытия unit-тестами. |
| Swagger | UI для чтения API-документации. |
| OpenAPI | Машиночитаемый формат описания API. |
| PostHog | Продуктовая аналитика. |
| Resend | Сервис отправки email. |
| Magic link | Вход по ссылке из письма без пароля. |
| Google One Tap | Быстрый вход через всплывающий Google prompt. |
| Telegram preauth | Промежуточный безопасный токен между Telegram callback и NextAuth session. |
| Identity | Внешний способ входа пользователя: Google, email, Telegram. |
| `user` | Внутренний пользователь сайта. |
| `books` | Таблица каталога книг. |
| `signup_books` | Таблица записей пользователя на книги. |
| `book_priorities` | Таблица порядка книг у пользователя. |
| `book_submissions` | Таблица предложенных книг. |
| `notification_queue` | Очередь будущих email-уведомлений. |
| CI | Continuous Integration: автоматические проверки качества. |
| E2E | End-to-end тесты, которые проверяют сценарии в браузере. |
| Cron | Запланированный автоматический вызов endpoint-а. |
