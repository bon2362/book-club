Направление 4 — Success State после отправки
Два варианта подтверждения. После отправки форма заменяется состоянием успеха.
A — Inline в модальном окне
✅
Заявка принята!
Мы рассмотрим её в ближайшее время.
Вы получите email когда статус изменится.
B — Toast / баннер сверху
✅
Заявка принята!
Уведомление придёт на email при смене статуса
Вариант A создаёт финальную точку — пользователь точно понимает что всё завершено. Вариант B менее навязчив, позволяет сразу вернуться к каталогу.
Направление 5 — Валидация и состояния полей
Inline валидация при потере фокуса, loading state кнопки, состояние фокуса.
Состояния полей
⚠ Это поле обязательно
✓ Выбрано
Слева — активная кнопка, справа — loading state
Валидация срабатывает при потере фокуса (onBlur), не при каждом нажатии клавиши. Loading state блокирует кнопку повторной отправки.
